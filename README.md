FakeKart is a React-based e-commerce frontend application that demonstrates dynamic product rendering, client-side routing, and shared cart state management. The project focuses on building a responsive shopping flow using modern React patterns and a public product API.




This project is actively being improved. Some features are still under refinement.

Key Features
1) Dynamic product listing using a public API
2) Responsive UI built with Bootstrap
3) Page-level navigation using React Router
4) Shared cart state management using React Context API
5) Lazy loading and code splitting with React.lazy for improved performance

Cart & State Management
The cart functionality is managed using React Context to maintain shared state across components. This allows products to be added, viewed, and managed consistently throughout the application without excessive prop drilling.
